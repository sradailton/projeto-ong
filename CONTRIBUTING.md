# Guia de Contribuição

Obrigado por considerar contribuir com a ONG Connect Platform! 

## Como Contribuir

### Reportando Bugs

1. Use o template de issue
2. Descreva o comportamento esperado vs atual
3. Inclua steps para reproduzir
4. Adicione screenshots se aplicável

### Sugerindo Melhorias

1. Explique a melhoria proposta
2. Descreva o caso de uso
3. Inclui exemplos se possível

### Submetendo Pull Requests

1. Siga o Git Flow
2. Use Conventional Commits
3. Atualize a documentação
4. Adicione testes se aplicável

## Padrões de Código

### HTML
- Use elementos semânticos
- Implemente ARIA labels quando necessário
- Mantenha hierarquia de headings lógica

### CSS
- Siga o sistema de design
- Use variáveis CSS
- Mantenha especificidade baixa

### JavaScript
- Use ES6+ features
- Siga padrões modular
- Documente funções complexas
